Bludit CMS | Histórico

Visão Geral

Com Bludit, crie seu próprio site ou blog em segundos.
Sistema rápido, seguro e de arquivo simples (do termo em inglês flat-file CMS).

Não requer banco de dados !
Não gaste em hospedagem cara, hospedagem compartilhada é o suficiente devido a sua leveza.

Linha do tempo

2015

  • Setembro - já implementado instalador que solicita usuario senha e email; na v0.1 já tinha a maioria das funcionalidades: criação de página estática e normal. Usuários: administrador e editor, URL amigável, filtragem para postagem, páginas e tags e utilizava o MarkDown como padrão para o editor; disponível em 9 idiomas; 1° tema de nome Pure, e 6 plugins (Disqus, Maintenance Mode, Open Graph, Page list, Simple MDE e Tag list); na versão 0.2 foi introduzido o idioma Português Brasileiro.
  • Outubro - A estrutura das pastas eram: admin, content, kernel, languages, plugins e themes; na versão 0.4 beta 1 recupera a senha via email e adição de página pai e filha (apenas no Páginas), além do item posição, adição do plugin Google Tools; na v0.5 melhorias no código de acesso via email e no sistema de página, atualização do plugin SimpleMDE e JQuery.
  • Novembro - na v0.6 b1 foi adicionado o novo sistema de carregamento de imagem e suporte para o SimpleMDE, e adicionado o TinyMCE; melhorias no gerenciador de usuario e no instalador; na v0.6.1 e 6.2 suporte para built in server, e melhoria para servidores web windows, formatos de datas para posts, suporte para o Parsedown extra, e novo tema Future Imperfect, imagem de perfil de usuario melhorada, adicionado o plugin About; movido a pasta admin para kernel; o tema padrão Pure e a interface do painel de controle já estava com a aparencia melhor que a 0.4; na versão 0.7 beta1 foi habilitado o TinyMCE como editor padrão, melhorado token CSRF e suporte a 15 idiomas.
  • Dezembro - Lançada versão 0.7 com melhorias no sistema de URL, escolha entre TinyMCE (padrão HTML) ou SimpleMDE (markDown); token de segurança aprimorado CSRF; painel de administração mais responsivo para celulares e tablets.

2016

  • Janeiro - na versão 1, introduzido o arquivo metadata.json na pasta bl-plugins e bl-themes ('inserido bl-' nas pasta padrões), melhoria no perfil de usuario e imagem do perfil, novo plugin posts recentes (lastest_posts), rss e sitemaps introdução dos hooks (ganchos), novo objeto User para postagens e páginas; imagem de capa para artigos e postagens além de novo sistema de upload, novo sistema de tag com preenchimento automático (auto complete), SimpleMDE como editor primário, novo filtro para mostrar o blog quando você setar a página como página padrão; novos temas Blogme e Future imperfect; compatibilidade com PHP 5.3; melhorias na paginação.
  • Fevereiro - na v1.1.2 diversas correções de bugs e tags ordenadas alfabeticamente.
  • Maio - na v1.3 beta 1 novo plugin API (não 100% funcional), correção de problemas em datas e tags, melhorias em páginas ordem/sort; novo tema Log, painel melhorado e mais responsivo, melhor performance no sistema.
  • Junho - Melhoria no plugin API, e melhoria na URL amigável; Atualização de idiomas, datas e tags, coração do sistema, css e melhorias nos plugins Sistema, Opengraph e SimpleMDE; Inserido na v1.4 a flag para identificar compatibilidade em temas e plugins (não lista quando não tem compatibilidade); Melhorias no core do Bludit.
  • Setembro - v1.5 beta 1 a 3 - dashboard alterado para barra lateral; API plugin e core do Bludit melhorado.
  • Outubro - Lançado v1.5 com as correções dos betas; Atualizados: tema Log, idiomas, core do Bludit, plugins API, instalador, painel de controle; Corrigido bugs: quando não existe página, ao fazer upload na imagem do perfil, nos plugins Sitemap e RSS e atualização de idioma.

2017

  • Maio - v1.6 beta 1 a 1.6.1; Adição: UUID e tempo de leitura para posts e páginas, novo layout para plugins e temas; Melhorias: sistema de imagens para Bludit, Paginação, mudança visual nas opções de novas páginas e posts; Atualizações: instalador, jQuery, ParseDown, dicionário de idioma; Diversos bugs corrigidos;
  • Junho - 1.6.2 pequenas correções; compatibilidade de tipos para plugins;
  • Julho - Bludit 2.0 alpha 1 a 3; Temas novos: Diego, Docs, Editorial, Kernel Panic, Massively, Phantom; Novos plugins: Fixed pages, Links, Menu; Inserido: Website ou blog order: por posição ou data; Em configuraçoes avançado URL filters foi removido 'blog' - consta /, tag e categoria;
  • Agosto - v2.0 beta 1; Alterações: Resposta do protocolo de código HTTP corrigida; Web hook corrigido para plugins, Paginator melhorado, Corrigido Posição do plug-in (apenas via codigo); Corrigido o regex para mudar em relação ao url absoluto de imagens carregadas; URL amigável do conteúdo quando você não define um título; Plugin Tinymce atualizado, este plugin usa o CDN do Tinymce; Imagem da capa externa; Melhora em temas e plugins; Correções de bugs; Adição do tema: Klean, Lingonberry; Adição dos plugins Categories, Twitter Cards, Ping.
  • Setembro
    • v2.b2 - Alterações: Remoção de tags HTML ao obter o título do conteúdo, se o título estiver vazio; Mudanças no painel, atualizações de CSS e JS; Mudanças no sistema de idioma, o local pode ser incluído no dicionário do idioma; O usuário pode definir a página 404 em Configurações avançadas, por padrão, a página 404 é gerada instantaneamente com um texto padrão; Melhora na API; Correções ao analisar o conteúdo do arquivo index.txt; Melhora no gerenciamento de conteúdo; Dicionários de idiomas atualizados; Tema Docs atualizado;
    • v2b3 - Filtro /blog/ adicionado; Dicionário de inglês atualizado; Melhora no roteador de URL; Correção de bugs;
    • v2b4 - Correções de bugs no roteador de URL, às vezes página retornada não encontrada; Dicionários de idiomas atualizados com algumas palavras que faltavam; Dicionário russo atualizado; Correções de bugs em diferentes temas; As páginas fixas agora são páginas estáticas; Plugin API atualizado, trabalhando na documentação.
    • v2b5 - Correções de bugs no roteador de URL; Plugin API, escrever funções, criar uma página, editar uma página e excluir uma página; Correções em páginas de gerenciamento, paginator.
    • v2b6 - As páginas estáticas são ordenadas por posição; API melhora; Dicionário do idioma inglês atualizado; Pequenas correções de bugs.
  • Outubro - v2 lançado e correção na v2.1; Dicionários de idiomas atualizados; Correções de funções de usuário no painel; Melhora ao obter o conteúdo do arquivo; Pequenas mudanças nas categorias de plugins; Páginas do plug-in renomeadas para o conteúdo mais recente do plugin; Adicionado a midia social Codepen; Novo tema da history; Postagem e páginas agora são conteúdo; Categorias Adicionadas; Notificações e Syslog para eventos; API renovada; Editor de Markdown SimpleMDE e Editor de HTML TinyMCE; Diversas atualizações de bugs.
  • Novembro - Atualizações de segurança para nova Tag e Categoria; Correção de bugs do painel de controle em smartphones, em paginação e temas, melhora no Helper de Filesystem, além de atualização de idiomas e algumas versões do PHP.
  • Dezembro - v2.1 função de 'lembrar me' na tela de login que armazena a sessão por 30 dias; Novo plugin Backup (crie, restaure e baixe); Nova classe para Categorias e Tags; Melhoria no suporte a datas multi-idiomas; Atualizações de idiomas e novo formato de JSON para autores; Bug corrigido, compatibilidade com PHP 7.2; Melhorias no core do Bludit.
