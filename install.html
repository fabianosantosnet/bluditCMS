Requerimentos e Preparação do Ambiente

Antes de tudo, é necessário preparar um servidor web com PHP 5.6 ou superior e os módulos PHP mbstring, gd, dom, json. No linux seria como: php*-mbstring php*-gd php*-xml php*-json libapache2-mod-php php-mcrypt onde no lugar do '*' é a versão do PHP.

Não é necessário banco de dados !

Bludit pode ser usado em quase todos webservers:

  • PHP Built-in web server
  • Nginx com módulo ngx_http_rewrite_module
  • Apache com módulo mod_rewrite
  • Lighttpd com módulo mod_rewrite
  • Hiawatha com rewrite rules
  • Para H2O veja o artigo H2O HTTP/2 web server and Bludit no fórum
  • IIS com módulo URL Rewrite, veja este post no fórum

Instalação do Bludit

Nesta seção será reunida informações de instalação do Bludit em diversas plataformas.

Documentação oficial

Conforme instrução de instalação da documentação oficial (en), após efetuar o download no site oficial, extraia na pasta de seu webserver, em seguida, abra via browser o caminho http://localhost/, para execução local na pasta raiz, ou http://localhost/bludit/ caso tenha extraido na pasta bludit. Substitua localhost pelo nome de domínio adequado.
