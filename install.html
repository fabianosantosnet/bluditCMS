Requerimentos e Preparação do Ambiente

Antes de tudo, é necessário preparar um servidor web com PHP 5.6 ou superior e os módulos PHP mbstring, gd, dom, json [see].

Não é necessário banco de dados !

Bludit pode ser usado em quase todos webservers:

  • PHP Built-in web server
  • Nginx com módulo ngx_http_rewrite_module
  • Apache com módulo mod_rewrite
  • Lighttpd com módulo mod_rewrite
  • Hiawatha com rewrite rules
  • Para H2O veja o artigo H2O HTTP/2 web server and Bludit no fórum
  • IIS com módulo URL Rewrite, veja este post no fórum

Instalar o PHP

Acesse o site oficial do PHP e saiba os procedimentos para instalação.

Pacotes

Caso prefira instaladores com tudo junto pode-se usar os WAMPs/LAMPs:

  1. WAMP (Windows, Apache, MySQL, PHP): Vertrigo Serv, WAMPServer (desinstale/não instale o banco);
  2. LAMP (Linux, Apache, MySQL, PHP): Veja o artigo de servidor web no Ubuntu/Debian ou mais customizado no Ubuntu (ignore o passo 2).
